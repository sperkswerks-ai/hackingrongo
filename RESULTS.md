# Quantum Hardware Results

Every row represents one IBM Quantum hardware submission made during this
project.  Job IDs can be looked up in the
[IBM Quantum Platform dashboard](https://quantum.ibm.com/jobs).

Files live in [`quantum_results/`](quantum_results/) and are named
`<run_key>_<timestamp>_<job_id>.json`.  Each file contains a
`hardware_provenance` block with the full backend calibration timestamp.
